# App Statement

An API that shows the user the most important Events/Holidays in Saudi Arabia.

# Feature List

### 

-  Create a new events.
-  Get months and display them to the user.
-  Get months by specific id.
-  Get events in each month and display them to the user.
-  update events.
-  Delete events.
